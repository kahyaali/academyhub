
## 🚀 Teknoloji Yığını

### Backend
- ASP.NET Core 8
- Entity Framework Core (Code-First)
- UnitOfWork & Repository Pattern
- AutoMapper
- JWT Authentication & Authorization
- RabbitMQ (Message Queue)
- Worker Service (Background Jobs)
- SQL Server (LocalDB)

### Frontend
- React.js
- React Router DOM
- Context API (State Management)
- Axios (HTTP Client)
- SweetAlert2 (Toast/Bildirim)
- React Icons

## ✅ Sistem Özellikleri

- Kullanıcı yönetimi (Admin, Instructor, Student)
- JWT ile güvenli kimlik doğrulama
- Şifre sıfırlama (JWT token ile)
- Kurs ve ders yönetimi
- Ders ekleme (Video, PDF, Metin)
- İlerleme takibi
- Yorum ve puanlama sistemi
- Admin onay sistemi
- RabbitMQ ile bildirim gönderimi
- Sertifika oluşturma (PDF)
- Çoklu para birimi desteği (TL, USD, EUR, GBP)
- Eğitmen kazanç takibi
- Ödeme raporları
